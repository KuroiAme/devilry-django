.. _email:

=====================================
Email sending in Devilry
=====================================

When does Devilry send email?
#############################################
When students:
 - adds a comment/uploads a file.

When examiners:
 - adds a comment/uploads a file.
 - changes the deadline.
 - gives a new attempt.
 - edit the grade/points.

When admins:
 - adds a comment/uploads a file.
 - changes the deadline.
